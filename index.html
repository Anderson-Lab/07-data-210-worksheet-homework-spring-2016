    In this homework you will create a database from scratch, insert, update, delete, and select entries from it.

    1. At bash (not posgres) type: createdb worksheet

    2. Then go into postgres and type: \connect worksheet

    3. Now you will have a clean database to work with.

    4. Define the following tables in postgres. The first one can be called Persons and the second can be called Courses.

    ID Name GPA
    1 Paul 3.8
    2 Aspen 3.2
    3 Sebastian 3.85
    4 Sarah 3.9
    5 Jill 2.5

    ID should be the primary key. Name should be a varchar with a maximum size of 10, and GPA should be a decimal.

    ID PID Name
    1 2 DATA 210
    2 2 CSCI 334
    3 3 CSCI 334
    4 4 DATA 210
    5 5 CSCI 470

    There is one foreign key and that is PID which references Courses. The primary key is ID, and the Name is varchar of size 10.

    For this question, you will need to master both the create table function and the insert function.

    1. Now write a simple select statement that returns the GPA and only the GPA for everyone.

    2. Now write a select statement that returns all the columns from the Courses table.

    3. Now write a select statement that returns the names of the people registered for DATA 210 (i.e., an inner join).

    4. Now write a select statement that returns the courses that everyone is registered for. If they aren't registered for any courses, you should till return the person as they are still a student.

    5. Now write a select statement that counts the number of people registered for DATA 210.

    6. Now write a select statement that returns the average GPA of the people registered for DATA 210.

    7. Now write a select statement that returns the average GPA of people registered for each class (i.e., group by).

    8. Now update the GPA of Aspen to be 3.5.

    9. Now delete Paul from the persons table.

    10. Now try to delete Aspen from the persons table.

    11. A few questions to think about: What is the purpose of an index? Do we have any indexes in our database? What is the command to create an index?
